
# Unnamed Text-based RPG

So, I tried to make a very simple text-based RPG.
This is the result.


What this game has:
- 5 files
- A map (duh...)
- A story (A very barebones one, it's kinda only there to incentivize beating the game, which doesn't even take that long)
- Simple combat (which is actually kinda boring)
- An inventory system
- A trading system
- Save/load
- LOTS of text


NOTE: This isn't _exactly_ an "RPG", there is no EXP or mana/spells system

(And I don't plan on adding them, since it's too late now)


To play this game, you'll need to install Python and run game.py

(Not shippable, sorry!)
